## The Lore
This was created as my Final year project during my Bachelor's degree. I followed the tutorial by clear code on youtube, and implemented some of my own additions like a new level, an ability to kill enemies, a pause menu and a leaderboard(stored only in the local database). I intend to add more levels and mechanics in the future(a double jump is on my mind)

## Controls
| Action | Key |
| :--- | :--- |
| Move Up | `W` or `Up Arrow` |
| Move Down | `S` or `Down Arrow` |
| Attack | `Spacebar` |
| Pause | `ESC` |

## Installation & Running Locally
To run this game on your own machine, you'll need Python installed. 

1. Clone the repository
   git clone [https://github.com/hellsguardianyt-cell/pyrates=game.git](https://github.com/hellsguardianyt-cell/pyrates-game.git)
   cd Pyrates-game
   
Create and activate a virtual environment:


Windows:

python -m venv venv
venv\Scripts\activate


Mac/Linux:

python3 -m venv venv
source venv/bin/activate

Install the dependencies:

pip install -r requirements.txt

Boot the game:

python main.py
